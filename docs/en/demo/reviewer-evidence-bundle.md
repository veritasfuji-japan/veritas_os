# Reviewer Evidence Bundle v1

The local Reviewer Evidence Bundle builder reproduces the reviewer-facing evidence artifacts generated by the GitHub Actions Reviewer Evidence Packet Validation workflow. It writes a deterministic local bundle containing the validation report, generated packet, golden fixture copy, schema copy, artifact manifest, manifest verification report, step summary, quickstart, and artifact index.

For external auditors and design partners who want to preview strict Evidence Bundle CLI results before receiving a real bundle, see the illustrative [Sample Evidence Bundle Verification Output](../validation/sample-evidence-bundle-verification-output.md).

This is useful for external reviewers, investors, enterprise stakeholders, and technical evaluators who want to reproduce the CI evidence bundle locally with one command.

## Usage

```text
python3 scripts/demo/build_reviewer_evidence_bundle.py
```

By default, the builder writes the bundle to:

```text
artifacts/reviewer-evidence-packet/
```

Optional output directory:

```text
python3 scripts/demo/build_reviewer_evidence_bundle.py --output-dir artifacts/reviewer-evidence-packet
```

The command prints a deterministic JSON summary to stdout. It exits `0` when generation and verification pass, and exits non-zero when a required generation, copy, or verification step fails.

## Expected output files

The bundle directory contains:

- `reviewer-evidence-packet-validation-report.json`
- `reviewer-evidence-packet-generated.json`
- `reviewer-evidence-packet-golden-fixture.json`
- `reviewer-evidence-packet-schema.json`
- `reviewer-evidence-artifact-manifest.json`
- `reviewer-evidence-artifact-manifest-verification-report.json`
- `reviewer-evidence-step-summary.md`
- `external-reviewer-quickstart.md`
- `external-reviewer-artifact-index.md`

## Local/offline boundary

The builder performs no live network calls and requires no credentials. It uses local VERITAS reviewer evidence scripts and checked-in artifacts only.

This local/offline bundle does not connect to live SaaS, IAM, IdP, SSO, customer directories, banks, sanctions systems, production approval workflows, or live audit stores.

It is not proof of live production deployment, audit certification, regulatory approval, third-party certification, legal advice, or production audit certification.
