# VERITAS OS Technical Proof Pack

The canonical English technical proof pack is
[`docs/en/validation/technical-proof-pack.md`](validation/technical-proof-pack.md).

For reviewer-facing Trusted Public Key Provenance artifacts in Reviewer Evidence
Packets, use the
[Reviewer Key Provenance Walkthrough](validation/reviewer-key-provenance-walkthrough.md),
the [Reviewer Handoff Guide](validation/reviewer-handoff-guide.md), and the
[Reviewer Handoff Sample Quickstart](validation/reviewer-handoff-sample-quickstart.md).

This forwarding entry point is not regulatory certification and is not completed
third-party audit approval. It does not create trust, does not replace
out-of-band public key trust, and does not re-run cryptographic verification.
