## 2. 意図された用途

### 2.1 VERITAS OS内での使用目的

- 安全ゲート付き意思決定パイプラインの中核推論エンジン
- 証拠に基づく多角的意思決定支援
- 議論（Debate）・批評（Critique）の生成
- リスク評価の参考情報生成

### 2.2 禁止用途

- 自律的な最終意思決定（人間レビューなし）
- 高リスク分野（医療・雇用・法執行等）での単独判断
- サブリミナル操作・社会スコアリング（EU AI法第5条）
- 生体認証による個人識別
- 予測的取締（predictive policing）

## 3. 既知の制約

| 制約カテゴリ | 詳細 |
|-------------|------|
| **ハルシネーション** | 事実でない情報を自信を持って生成する場合がある。VERITAS OSのEvidence検証・Critiqueモジュールで緩和。 |
| **非決定性** | 同一入力でも異なる出力を返す場合がある。温度パラメータ制御・Replay機能で追跡可能。 |
| **知識のカットオフ** | 訓練データのカットオフ日以降の情報を持たない。Web検索モジュールで補完。 |
| **バイアス** | 訓練データに含まれるバイアスを反映する可能性がある。バイアス評価参照。 |
| **言語性能差** | 英語に最適化されており、日本語等の非英語言語では性能が低下する場合がある。 |
| **トークン制限** | コンテキストウィンドウに制限がある。長大な入力は切り詰められる。 |
| **専門領域の限界** | 法律・医学・工学等の高度専門知識は必ずしも正確でない。 |
