# VERITAS OS v2.0.0 — Auditable Decision OS for LLM Agents

> 「安全で一貫性があり、監査可能な意思決定OS」

VERITAS OS は LLM（OpenAI API 等）をラップする Proto-AGI フレームワークです。
v2.0.0 では、コアサブシステムの堅牢化、Memory API の大幅改善、境界チェッカーの強化、デプロイメント基盤の整備など、100 以上のコミットによる改善が含まれます。

## ✨ v1.0.0 からの主な変更点

### Memory API の強化
- MemoryStore 互換ヘルパーの全面リファクタリング
- MemoryStore ロック・パッチ互換性の修正
- Memory API 例外境界の厳格化
- Memory API 障害分類の精緻化
- メモリ検索・蒸留ヘルパーのリファクタリング
- メモリ消去コンプライアンスフローの整備
- MemoryOS プランナーサマリーフォーマット改善

### 境界チェッカーの強化
- アーキテクチャ境界ドキュメントとの整合性チェック
- Markdown パース堅牢化（CRLF 対応、弾丸パース改善）
- 重複セクション検出
- 無効エンコーディング処理
- ドキュメントドリフトの CI 統合
- JSON 修正メタデータ追加

### デプロイメント基盤
- デプロイメント環境テンプレートスモークチェック追加
- デプロイメント環境フェイルオープンチェックの堅牢化
- 本番プロファイルスタートアップ警告の強化
- CI でのデプロイメントスモークチェック強制

### セキュリティ・信頼性
- TrustLog 集約バリデーションの堅牢化
- 認証フェイルオープンプロファイルチェックの強化
- TrustLog 集約ステータスのメトリクス公開
- 高優先度監査修正の回帰テストカバレッジ追加

### FUJI Gate
- FUJI ポリシー互換ローダーのリファクタリング
- FUJI ヘルパーユーティリティのリファクタリング
- FUJI ローダー CI 互換性修正

### ドキュメント・運用
- README ベータポジショニング更新
- コア拡張ポイントと運用ランブックの明確化
- 日本語精密コードレビューレポート追加
- 能力プロファイルランブックガード追加

### CI/CD

## 📦 インストール

```bash
git clone https://github.com/veritasfuji-japan/veritas_os.git
cd veritas_os
pip install -r requirements.txt
cp .env.example .env  # API キーを設定
python -m veritas_os.main
```

Docker Compose での起動:
```bash
docker compose up --build
```
